# **App Name**: Profit Pilot

## Core Features:

- UI Design: Neumorphic UI: A sleek, modern interface with a neumorphic design, providing an intuitive user experience.
- Real-time Metrics: Real-time metrics calculation, automatically calculating gross profit, breakeven ROAS, and ad budgets based on user inputs.
- Funnel Visualization: Funnel visualization: Display a dynamic inverted funnel, giving the user a picture of how their advertising spend will be allocated.
- AI Recommendations: AI-Powered Recommendation Engine: Provides suggestions to the user based on the key performance indicators
- Workflow Automation: Workflow automation "tool": Generate an n8n workflow configuration for ads automation. The generated code will attempt to automate budget optimization based on performance alerts and reporting for various advertising platforms, offering users an integration pipeline.
- Data Persistence: Data Persistence: Enables users to save their metrics and calculations locally for future access and reference.
- Automatic Titles: AI copywriting tool: Use AI to automatically generate titles for sections of the UI (in English).

## Style Guidelines:

- Primary color: HSL(187, 100%, 50%) / RGB(0,245,255) for a vibrant and modern aesthetic, reflecting a clean style, referencing the brand's neon highlights in dark mode.
- Background color: HSL(187, 20%, 10%) / RGB(20, 59, 71) - A dark and desaturated teal provides contrast.
- Accent color: HSL(157, 71%, 48%) / RGB(26, 199, 123) will highlight calls to action
- Body and headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward aesthetic.
- Use minimalistic line icons for key actions and categories to maintain a clean and modern aesthetic.
- Maintain a clean and structured layout, utilizing grid and flexbox for responsive design.
- Subtle transitions and animations for user interactions. For instance, subtle scaling animations for buttons, loading animations for metric calculations, and smooth transitions between sections.
